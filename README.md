#  Automated CI/CD Pipeline for Web Application

## Technologies Used

- Jenkins
- GitHub
- Docker
- AWS EC2
- Flask
- Shell Scripting

## Project Workflow

Developer
↓
GitHub
↓
Webhook
↓
Jenkins Pipeline
↓
Docker Build
↓
Docker Container
↓
AWS EC2 Deployment

## Features

- Automated Build
- Docker Image Creation
- Automatic Deployment
- GitHub Webhook Integration
- Shell Script Automation
- Docker Cleanup

## Run Locally

```bash
pip install -r requirements.txt
python app.py
```

Application URL:

http://localhost:5000
